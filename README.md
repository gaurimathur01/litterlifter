# Garbage Management System

## Overview
The Garbage Management System is a full-stack application designed to improve urban cleanliness and resource management by enabling efficient reporting, task assignment, and scheduling for garbage collection. This system involves different user roles such as residents, garbage collectors, and administrators, each having specific functionalities to ensure a smooth operation.
- **Secure Login System**: Ensures secure login for residents, garbage collectors, and administrators.
- **Role-Based Access Control**: Provides appropriate access to features based on user roles.
